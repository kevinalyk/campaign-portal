Script Embed Method

This method uses a script tag and a div with a data attribute:

<script src="https://www.campaignportal.ai/api/embed-script"></script> <div id="campaign-chatbot" data-campaign-id="67c796fa5be54c590a823566"></div>

iFrame Embed Method

This method uses an iframe to embed the chatbot:

<iframe src="https://www.campaignportal.ai/embed/chatbot?campaignId=67c796fa5be54c590a823566" width="100%" height="600px" style="border: none; max-width: 400px;" allow="clipboard-write" ></iframe>

Troubleshooting:

  • If the chatbots don't appear, check that your campaign ID is correct
  • Ensure your campaign is active in the Campaign Portal dashboard
  • Check browser console for any errors
  • Verify that your domain allows embedding (CSP settings)
